## 🎯 应用功能

这个 Tauri 应用现在包含两个主要功能：

### 1. 📝 Reminders（提醒应用）
完整的提醒管理系统，包括：
- **Smart Lists（智能列表）**
  - Today：今天的提醒
  - Scheduled：所有已计划的提醒
  - Flagged：已标记的重要提醒
  - All：所有提醒
  
- **自定义分类**
  - Work（工作）
  - Personal（个人）
  - Shopping（购物）
  - Health（健康）
  - Other（其他）
  - 可以添加更多自定义分类

- **功能特性**
  - 添加/编辑/删除提醒
  - 设置提醒时间
  - 标记完成状态
  - 重要事项标记（旗帜）
  - 重复提醒（一次、每天、每周、每月、每年）
  - 详细信息面板
  - 实时同步（跨窗口）
  - 调试模式

### 2. 💾 Disk Scanner（磁盘扫描器）
磁盘空间分析工具，包括：
- 扫描任意文件夹
- 可视化大小显示
- 文件夹树状展开/折叠
- 深度限制控制
- 权限错误处理
- 按大小排序
- 快速访问主目录和系统根目录

## 💡 使用方法

### 切换视图
1. 应用顶部有两个标签页：
   - 📝 **Reminders** - 提醒应用（默认）
   - 💾 **Disk Scanner** - 磁盘扫描器

2. 点击对应的标签即可切换功能

3. 每个视图的数据都是独立的，切换后不会丢失

### Reminders 使用
1. **添加提醒**：
   - 在顶部输入框输入标题
   - 按 Enter 或点击其他地方自动保存
   - 可以设置时间、分类、重复频率

2. **管理提醒**：
   - 点击圆圈标记完成
   - 点击旗帜图标标记重要
   - 双击提醒打开详情面板

3. **分类管理**：
   - 左侧边栏显示所有分类
   - 点击分类查看该分类的提醒
   - 可以添加自定义分类

### Disk Scanner 使用
1. **扫描文件夹**：
   - 输入路径或点击 "Home"/"Roots" 快速选择
   - 可选：启用深度限制（推荐3-5层）
   - 点击 "Scan" 开始扫描

2. **查看结果**：
   - 点击文件夹展开查看内容
   - 查看大小条和百分比
   - 按大小排序（最大的在前）

## 🔧 技术栈

- **Frontend**: Vue 3 + TypeScript + Vite
- **Backend**: Rust + Tauri 2
- **Database**: SQLite（提醒数据）
- **Styling**: CSS3（原生样式）

## 📝 注意事项

1. **数据存储**：
   - Reminders 数据存储在本地 SQLite 数据库
   - Disk Scanner 是即时扫描，不保存数据

2. **性能提示**：
   - 扫描大型文件夹时建议使用深度限制
   - 提醒应用支持实时同步

3. **权限**：
   - 某些系统文件夹需要管理员权限
   - 权限错误会显示 ⚠️ 标记，但不会中断扫描

## ❓ 常见问题

**Q: 看不到提醒内容？**
A: 确保点击了顶部的 "📝 Reminders" 标签，默认会显示 Today 列表。

**Q: 如何添加提醒？**
A: 在提醒列表顶部的输入框输入标题，按 Enter 即可。

**Q: 磁盘扫描很慢？**
A: 启用"限制深度"选项，设置为2-3层可以显著加快扫描速度。

**Q: 如何切换功能？**
A: 点击顶部导航栏的标签即可切换。
